Install docker on windows

Wifi car 0

Communate with ESP8266 via PL2303HXD

Use arduino as power supply to give enough energy to ESP8266

【Arduino】3.3V →【ESP8266】VCC
【Arduino】3.3V →【ESP8266】CH_PD
【Arduino】GND  →【ESP8266】GND
【Arduino】GND  →【PL2303HXD】GND   
【PL2303HXD】TXD→【ESP8266】URXD
【PL2303HXD】RXD→【ESP8266】UTXD

then you can talk to esp8266 by AT command.
